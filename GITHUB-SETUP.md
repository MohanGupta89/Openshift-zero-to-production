---

## Step 1 — Create GitHub Account (if you don't have)
```
Go to: github.com
Click: Sign Up
Use your email
Done
```

## Step 2 — Create New Repository on GitHub
```
1. Login to github.com
2. Click "+" top right → "New repository"
3. Name: openshift-zero-to-architect
4. Description: Complete OpenShift guide from zero to architect
5. Set to: Public (so others can find you — good for career!)
6. DO NOT initialize with README (we have our own)
7. Click "Create repository"
```

## Step 3 — Push This Content to GitHub
```bash
# Install git if not present
sudo yum install -y git

# Configure git
git config --global user.name "Your Name"
git config --global user.email "your-email@gmail.com"

# Go to your repository folder
cd openshift-zero-to-architect

# Initialize git
git init

# Add all files
git add .

# First commit
git commit -m "Initial commit: Complete OpenShift Zero to Architect guide"

# Add your GitHub repo as remote
# Replace YOUR_USERNAME with your GitHub username
git remote add origin https://github.com/YOUR_USERNAME/openshift-zero-to-architect.git

# Push to GitHub
git push -u origin main
```

## Step 4 — Add to LinkedIn
