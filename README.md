## ⚡ Quick Start (Do This First — Free, No Installation)

```bash
# Step 1: Get free OpenShift cluster in browser (takes 2 minutes)
# Go to: developers.redhat.com/developer-sandbox
# Sign up free → Get real OpenShift cluster in browser

# Step 2: Install oc CLI on your laptop
# Download from: mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/

# Step 3: Login to your sandbox
oc login --token=<your-token> --server=<your-server>

# Step 4: Check you are connected
oc whoami
oc get projects
```
